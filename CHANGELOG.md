# Changelog

本项目遵循语义化版本号。

## Unreleased

- 便签改为无标题结构，旧标题及旧版全局颜色、全局置顶字段不再读取或保存。
- 添加框移动到待办列表末尾并随列表滚动；长待办会在便签边缘自动换行。
- 每张便签通过标题栏左侧的九色面板独立配色；新便签始终为淡黄且默认不置顶。
- 标题栏新增新建和单张隐藏按钮；隐藏后不进入任务栏，单击通知区域图标恢复全部。
- 设置窗口移除“便签”页，只保留常规和关于。
- 浅色主题正文使用 `DESIGN.md` 的黑色 Ink，深蓝主题使用白色 Inverse Ink。

## 0.2.3 - 2026-07-06

- 设置窗口改为基于 Windows 工作区与 Tk DPI 比例计算尺寸，不再依赖单一 `780×660` 环境。
- 设置卡片使用内容驱动的最小高度；空间不足时页面可滚动，避免字体缩放后裁切。
- 修复嵌套卡片的自然高度未传递到滚动容器的问题，避免内容已经需要更多空间但滚动条仍不出现。
- 导航、设置行、更新卡、开源链接和主题网格会按可用宽度自动重排。
- 设置布局回归扩展到 800×560 至 3840×2080、100% 至 200% DPI，并同时覆盖中英文。

## 0.2.2 - 2026-07-05

- 兼容 GitHub 自动将 Release 安装包文件名中的空格改为点号的规则，确保检查更新能识别正式安装包。
- 下载后仍保存为易读的本地安装包名称，并使用 Release 内 `SHA256SUMS.txt` 的原始名称进行等价校验。

## 0.2.1 - 2026-07-05

- 修复启用 DPI 感知后，`780×660` 设置窗口的便签页与关于页底部卡片被裁切的问题。
- 移除意义不明确的“轻量桌面模式”，新增中文与英文界面切换。
- 便签颜色和置顶设置会立即应用到已有便签，并继续作为新便签默认值。
- 恢复带圆形滑块的抗锯齿开关控件。
- “关于”页可检查 GitHub 正式更新，自动下载 Release 安装包与校验文件，SHA-256 验证通过后启动安装向导。
- “关于”页移除更新来源卡片，改为简洁的开源仓库链接。
- 新增 DPI 布局回归、更新包验证、语言切换和已有便签同步测试。

## 0.2.0 - 2026-07-05

- 设置页改为白色编辑框架、单色大面板和黑色胶囊导航的现代视觉系统。
- 保留淡黄与黯白，并新增青柠、丁香、奶油、淡粉、薄荷、珊瑚和深蓝便签。
- 深蓝便签自动使用高对比度浅色操作图标。
- 单击通知区域图标立即唤起全部便签；设置窗口从任务栏激活时同步抬起便签。
- 唤起窗口不会改变每张便签原有的置顶状态。
- 构建缓存会验证当前 Python ABI，不兼容时自动刷新对应依赖。

## 0.1.0 - 2026-07-04

首个公开预览版本：

- 多张 Todo 便签，支持完成、编辑和删除待办。
- 淡黄与黯白两种便签颜色。
- 便签置顶切换、窗口位置与尺寸持久化。
- 通知区域菜单、单实例运行和开机自启。
- 便签窗口不占用任务栏，设置窗口按需显示任务栏入口。
- 本地 JSON 存储、备份与损坏恢复。
- Windows NSIS 安装、卸载和自动化发布流程。
